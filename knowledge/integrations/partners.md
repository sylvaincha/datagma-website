## Clay

### Pourquoi une page Clay ?
- Clay est devenu le standard de l'outbound moderne
- Datagma est un provider natif Clay
- Énorme base d'utilisateurs Clay qui cherchent des providers téléphone

### Structure de la page
1. **Hero** : "Use Datagma in Clay to find mobile numbers nobody else has"
2. **Pourquoi Datagma dans Clay** : spécialiste téléphone mobile, taux de match 70-77%
3. **Comment ça marche** : 
   - Créer un compte Datagma → récupérer sa clé API
   - Dans Clay : Add Enrichment > Datagma > coller la clé
   - Choisir le type d'enrichissement (phone, email, full)
4. **Cas d'usage** : waterfall phone enrichment dans Clay
5. **Pricing** : coût par enrichissement

### Messages clés
- "Native Clay integration — set up in 2 minutes"
- "Fill the phone number gaps in your Clay tables"
- "77% mobile match rate in India, 69% in UK"

### Structure de la page
1. **Hero** : "Apollo for emails + Datagma for phones — the winning combo"
